# ComfyUI JSON Reader for After Effects

A ScriptUI panel that extracts and displays generation parameters from ComfyUI-generated PNG images directly inside After Effects.

![JSON Reader Panel](https://raw.githubusercontent.com/ckonteos80/Adobe-After-Effects-ComfyUI-Panels/main/Screenshots/JsonReaderPanel.jpg)

## Features

- **Instant Metadata Reading**: Fast PNG chunk parsing extracts ComfyUI data in milliseconds
- **Parameter Extraction**: Automatically parses seed, steps, CFG, sampler, scheduler, size, denoise, model, and prompts
- **Flux Support**: Reads parameters from Flux.1 and Flux.2 workflows (`RandomNoise`, `KSamplerSelect`, `Flux2Scheduler`, `CLIPTextEncodeFlux`, `UNETLoader`)
- **API Prompt Access**: View, copy, and export the full API-format prompt JSON for regeneration
- **Layer Integration**: Read metadata directly from selected footage layers in the active composition
- **Auto-Refresh**: Updates automatically when you select a different layer
- **No External Dependencies**: Pure ExtendScript ‚Äî no additional tools required

## Installation

1. In After Effects, go to **File ‚Üí Scripts ‚Üí Install ScriptUI Panel...**
2. Select `ComfyUI_JsonReader.jsx` and click Open
3. Restart After Effects
4. Open the panel via **Window ‚Üí ComfyUI_JsonReader.jsx**

> **Note:** Enable script access if prompted:  
> **Edit ‚Üí Preferences ‚Üí Scripting & Expressions ‚Üí Allow Scripts To Write Files And Access Network**

## Requirements

- Adobe After Effects CC 2018+
- PNG images generated by ComfyUI with embedded metadata

## Usage

1. **Import a ComfyUI PNG** into your project and add it to a composition
2. **Select the footage layer** in the timeline
3. **Click "From Selected Layer"** ‚Äî or use **Browse PNG** to open a file directly
4. All detected generation parameters are displayed immediately
5. Use **Copy API Prompt** or **Save API Prompt** to export the prompt JSON

### Controls

- **From Selected Layer** ‚Äî reads metadata from the currently selected PNG footage layer
- **Browse PNG** ‚Äî opens a file picker to select any PNG
- **‚ü≥ Refresh** ‚Äî re-reads the currently selected layer
- **Copy API Prompt** ‚Äî copies the full API JSON to clipboard
- **Save API Prompt** ‚Äî saves API JSON to a file

## Supported Node Types

Parameters are filled in priority order ‚Äî `KSampler`/`KSamplerAdvanced` wins over fallback nodes for the same field.

### Sampler nodes
| Node | Fields extracted | Workflows |
|------|-----------------|-----------|
| `KSampler` / `KSamplerAdvanced` | seed, steps, cfg, sampler_name, scheduler, denoise | Standard SD / Flux.1 |
| `KSamplerSelect` | sampler_name | Flux.2 custom-advanced pipeline |
| `RandomNoise` | noise_seed ‚Üí Seed | Flux.2 (seed lives here, not in the sampler) |
| `Flux2Scheduler` | steps, width, height | Flux.2 |

### Latent / dimension nodes
| Node | Fields extracted | Workflows |
|------|-----------------|-----------|
| `EmptyLatentImage` | width, height | SD1.5, SDXL |
| `EmptySD3LatentImage` | width, height | SD3, Flux.1, Qwen Image |
| `EmptyFlux2LatentImage` | width, height | Flux.2 |
| `LatentUpscale` / `LatentUpscaleBy` | width, height | Upscale workflows |

### Prompt nodes
| Node | Fields extracted | Workflows |
|------|-----------------|-----------|
| `CLIPTextEncode` | text ‚Üí Positive / Negative Prompt | SD, Flux.2, Qwen Image |
| `CLIPTextEncodeFlux` | clip_l ‚Üí Positive Prompt | Flux.1 |

### Model nodes
| Node | Fields extracted | Workflows |
|------|-----------------|-----------|
| `CheckpointLoaderSimple` / `CheckpointLoader` | ckpt_name ‚Üí Model | SD / Qwen |
| `UNETLoader` | unet_name ‚Üí Model | Flux.1, Flux.2 |

## Troubleshooting

### "No text chunks found in PNG file"
- Image was not generated by ComfyUI, or metadata was stripped during export
- Use ComfyUI's **Save Image** node ‚Äî do not screenshot or re-export the file

### "No ComfyUI workflow data found"
- PNG has text chunks but they are not ComfyUI metadata
- May be from an older ComfyUI version before metadata embedding was added

### "No API Prompt loaded"
- Image only contains the UI-format workflow chunk, not the API prompt chunk
- Parameters can still be extracted from the workflow chunk as a fallback

### "Please select a PNG footage layer"
- Selected layer must be a footage item backed by a `.png` file that exists on disk
- Solids, shape layers, and text layers are not supported

## Tips

- **Auto-refresh** activates when you click different layers ‚Äî no need to keep hitting Refresh
- **Copy API Prompt** gives you a JSON you can POST directly to ComfyUI's `/prompt` endpoint
- **Save API Prompt** lets you build a library of reusable generation settings
- Works alongside the [Text2Image Panel](../Text2Image/) ‚Äî use JSON Reader to inspect a result, then adjust settings in Text2Image for a new generation
